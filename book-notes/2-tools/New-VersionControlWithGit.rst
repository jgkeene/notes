4 - Git Objects and Index
==========================

Objects stored in `repo/.git/objects/##/#######################` 

Viewing objects ::

  git cat-file -p
  git ls-files --stage



Index stored at `.../.git/index`

Viewing index::

  git ls-files --stage
  git status
  git diff --cached
